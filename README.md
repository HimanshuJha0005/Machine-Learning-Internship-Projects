# Machine Learning Internship Projects

This repository contains end-to-end Machine Learning projects developed using Python, Scikit-Learn, and Pandas in Google Colab.

---

##  Projects Summary

### 1. Movie Genre Classification
* **Objective:** Predict movie genres based on text summaries/plots.
* **Algorithm:** Multinomial Naive Bayes with TF-IDF Vectorization.
* **Key Focus:** Text preprocessing, Feature Extraction, Multi-class Classification.

### 2. SMS Spam Detection
* **Objective:** Classify incoming SMS messages as SPAM or HAM (Legitimate).
* **Algorithm:** Multinomial Naive Bayes with TF-IDF Vectorizer.
* **Accuracy:** ~97.3%
* **Key Focus:** Text Cleaning, Precision & Recall evaluation.

### 3. Credit Card Fraud Detection
* **Objective:** Detect fraudulent credit card transactions on imbalanced datasets.
* **Algorithms Benchmarked:** Logistic Regression, Decision Tree, Random Forest.
* **Best Model:** Random Forest Classifier (**93.31% Accuracy**, F1-Score: 0.87).
* **Key Focus:** Handling Imbalanced Data (Under-sampling), Feature Scaling, Model Benchmarking.
